Stopwatch-Web-App

A sleek, user-friendly, and fully responsive Stopwatch Web Application developed during my internship at SkillCraft Technology 🚀. This project is designed to help users accurately track time intervals with an intuitive interface, smooth functionality, and a seamless user experience.

🌟 Key Features

✔️ Start, Pause & Reset Stopwatch

✔️ Record and Display Lap Times

✔️ Track Multiple Laps Efficiently

✔️ Light & Dark Theme Toggle

✔️ Accurate Real-Time Time Tracking

✔️ Clean & Interactive UI

✔️ Fully Responsive Design

✔️ Smooth User Experience

🛠️ Technologies Used

HTML5

CSS3

JavaScript

🎯 Project Goal

The main objective of this project is to provide a simple and efficient time-tracking solution with lap management and theme customization features, while enhancing user experience through a modern and responsive interface.
